Welcome to the Number Recognizer!

Created by Sid Ijju

In this app, you will be able to draw a number and have a machine learning algorithm attempt to recognize it!

The machine learning algorithm was created using the MNIST Digit Recognition Dataset and Convolutional Neural Networks.

To run the algorithm, please write a sample digit, and then hit Submit to have the algorithm classify it.

Otherwise, you can clear the box and redraw any digit.
